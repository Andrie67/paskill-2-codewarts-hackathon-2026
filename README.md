# PaSkill Mobile Prototype

PaSkill is a mobile-first digital worker credibility platform connecting verified blue-collar workers with employers.

## Features
- **Worker Flow**: Sign up, upload proofs, generate a Digital Worker Passport, browse jobs.
- **Employer Flow**: Post jobs, browse verified workers, access interested candidates, and send direct messages.

## Tech Stack
- React
- Vite
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Zustand (Session Storage)

## Setup
```bash
npm install
npm run dev
```
